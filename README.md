# tomurcuk
# Sahibinden Hackathon 

## Project

A need adding-listing-following tracking platform that brings together the people in need with the public and philanthropists in the disaster area.

## Technologies

- MVVM
- Jetpack Navigation
- ViewModel
- LiveData
- Firestore
- SharedPreferences
